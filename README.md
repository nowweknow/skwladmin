# SKWL ADMIN

## ENVIRONMENT

env variables should be configured in root `.env` file:

```
REACT_APP_BASE_URL=''
```
## Installation

```bash
$ npm install
```

## Running the app
```bash
npm run start
```
## Accesss to admin panel
login: admin

password: root

## Route to admin panel

http://localhost:3000/admin-panel
